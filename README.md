
# Happy House

🔔 본 프로젝트는 **삼성 청년 SW 아카데미** 1학기 관통 프로젝트 결과물입니다

<br><br>



## **개요**

- 부동산 매물의 다양한 서비스 제공을 목적으로 매물 실거래가, 최신 매물 정보, 실거주자 리뷰, 친구 기능 등을 제공

<br><br>

## **프로젝트 기간**

- 2022년 12월 17일 ~ 2022년 11월 24일 (8일)

<br>
<br>

## **성과**

🔔 삼성 청년 SW 아카데미 1학기 프로젝트 우수상 수상
