# YoBangCok

<img src="./assets/YoBangCok_logo-003.png">

## 🏠 프로젝트 개요

부동산 매물의 다양한 서비스 제공을 목적으로 공공데이터 기반 매물 실거래가 조회, 실시간 매물 등록 및 조회, 게시글, 댓글, 좋아요 기능 등을 제공합니다.

![YoBangCok](./assets/yobangcokgif.gif)

<br>
